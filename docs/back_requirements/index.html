Backend requirements

We are using netcoreapp3.0 as base, main dependencies are Microsoft.AspNetCore.Mvc.NewtonsoftJson for json parsing, Microsoft.ApplicationInsights.AspNetCore to collect metrics, MSTest.TestFramework for tests and NSwag.AspNetCore for swagger API doc
